Bluemarlin

Automated search tool for wraeclast items

View On GitHub

    Bluemarlin is a fan made open source (GPLv2) items search tool for the Free to Play ARPG online game Path of Exile

    Features:

    1. User define search files (elastic search queries)
    2. Auto search for search files listed under Durian directory
    3. Fully customizable search result renderer via HTML5
    4. Always on top Widget for notification
    5. Default search files and renderer
